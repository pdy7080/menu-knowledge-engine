# 수익 모델

## 1. B2B SaaS (핵심 수익원)

### 가격 벤치마크 — 한국 소상공인 지불 의향: 월 3~10만 원
| 기존 서비스 | 월 가격 |
|---|---|
| 캐치테이블 | 3.3~9.9만 원 |
| 나우웨이팅 | 약 3.5만 원 |
| 테이블링 | 9.9만 원 |

### 권장 가격 체계

| 플랜 | 기능 | 월 가격 | 타겟 |
|---|---|---|---|
| **Free** | 메뉴 10개, 1개 언어, 워터마크 | 무료 | 체험/전환 유도 |
| **Basic** | 메뉴 50개, 3개 언어 | 1.9~2.9만 원 | 소규모 로컬 식당 |
| **Pro** | 메뉴 무제한, 5개 언어, AI 추천 | 4.9~6.9만 원 | 관광지 식당 |
| **Enterprise** | 다점포, API 연동, 맞춤 브랜딩 | 9.9~19.9만 원 | 프랜차이즈/호텔 |

### 핵심 경제성: 마진율 90% 이상
```
번역 원가: 100개 메뉴 × 5개 언어 = 약 $0.50~$1.00 (≈700원)
Basic 플랜 판매가: 월 19,000원
→ 마진율: 96%+
```

## 2. B2C 수익화

| 모델 | 내용 | 예상 수익 |
|---|---|---|
| 프리미엄 구독 | 알레르기 필터, 오프라인 저장, AI 맞춤 추천 | 월 $2.99~$4.99 |
| 식당 프로모션 광고 | 식당이 외국인 노출을 위해 비용 지불 | CPC/CPM |
| 예약 연계 커미션 | 캐치테이블, 마이리얼트립 등 | 예약당 5~20% |

디지털 접근성: 방한 외국인 **91.7%가 한국 앱 이용** → B2C 플랫폼 진입 가능

| API | 월 가격대 | 특징 |
|---|---|---|
| Spoonacular | $0~$149 | 서양 음식 중심 |
| Edamam | $19~$999 | 영양 정보 특화 |
| Nutritionix | $299~$1,850 | 대형 체인 DB |
| **한국 음식 특화 API** | **전무** | **← 시장 공백** |

### API 잠재 고객
| 고객군 | 대표 기업 | 니즈 |
|---|---|---|
| 배달앱 | 배민, 요기요, 쿠팡이츠 | 외국인 모드 메뉴 데이터 |
| 관광 플랫폼 | 트립어드바이저, 클룩, 마이리얼트립 | 식당 추천 콘텐츠 |
| 식당 예약 앱 | 캐치테이블 (가맹점 1만 개+) | 다국어 메뉴 정보 |
| 지도 서비스 | 네이버지도 (외국인 이용률 56.2%) | 식당 상세 정보 |
| 숙박/항공 | 항공사, 호텔 체인 | 투숙객 한식 가이드 |

### API 판매 채널
