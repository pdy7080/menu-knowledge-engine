# Executive Summary (핵심 요약)

> **AI/OCR 기반 한국 음식 메뉴판 자동 다국어 변환 + 문화적 맥락 설명 서비스**

## 왜 지금인가? (시장 타이밍)

| 팩트 | 수치 |
|---|---|
| 2025년 방한 관광객 예측 | **1,873만 명** (역대 최고) |
| 식도락 관광 참여율 | **73.2%** |
| 서울 식당 메뉴 오역률 | **34%** |
| 외국인 개선 요구 1위 | "다양한 외국어 지원" |
| 시장에 통합 서비스 | **없음** |

---

## 핵심 차별화

**"번역은 있지만, 설명은 없다"** → 우리가 채운다

1. **문화적 맥락 설명** — 단순 번역이 아닌 맛·역사·먹는 방법까지
2. **손글씨 OCR** — 로컬 식당의 비정형 메뉴판 인식
3. **알레르기 정보** — 외국어 제공 식당 거의 전무한 시장
4. **한국 음식 온톨로지 DB** — 시간이 지날수록 깊어지는 데이터 해자(moat)
